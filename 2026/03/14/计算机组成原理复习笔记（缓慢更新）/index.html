一.计算机系统概论

1.1 计算机系统的软硬件概念

  • 硬件：指计算机系统中实际存在的物理设备，如CPU、内存、硬盘、显示器等。
  • 软件：指计算机系统中运行的程序和数据，如操作系统、应用程序、数据库等。

"软件"的地位和作用已经成为评价计算机系统性能好坏的重要指标。
计算机软件可以分为两大类：系统软件和应用软件。

  • 系统软件：指操作系统、编译程序、汇编程序等，它们是计算机系统的基础，提供了计算机系统的运行环境和服务。
  • 应用软件：指用户在计算机上进行特定任务的程序，如文字处理、 spread sheet、数据库管理系统等。

1.2 计算机系统的层次结构

虚拟机器M4(高级语言机器):

  • 它是一种能够直接执行高级语言程序的机器。
  • 高级语言程序是一种人可读的程序，而机器指令是一种机器可读的程序。
  • 高级语言机器通过编译程序将高级语言程序转换为汇编语言程序或直接机器语言程序，然后执行这些程序。
    |
    |
    虚拟机器M3(汇编语言机器):
  • 它是一种能够直接执行汇编语言程序的机器。
  • 汇编语言程序是一种人可读的程序，而机器指令是一种机器可读的程序。
  • 汇编语言机器通过汇编程序将汇编语言程序转换为机器指令程序，然后执行这些程序。
    |
    |
    虚拟机器M2(操作系统机器):
  • 它是一种能够直接执行操作系统程序的机器。
  • 操作系统程序是一种人可读的程序，而机器指令是一种机器可读的程序。
  • 操作系统机器通过操作系统将应用程序转换为机器指令程序，然后执行这些程序。
    |
    |
    虚拟机器M1(机器语言机器):
  • 它是一种能够直接执行机器指令程序的机器。
  • 机器指令程序是一种机器可读的程序，而机器指令是一种机器可读的程序。
  • 机器语言机器通过直接执行机器指令程序来完成计算任务。
    |
    |
    微程序机器M0(微指令系统):
  • 它是一种能够直接执行微指令程序的机器。
  • 微指令程序是一种机器可读的程序，而微指令是一种机器可读的程序。
  • 微程序机器通过直接执行微指令程序来完成计算任务。

1.3 计算机组成和计算机体系结构

计算机体系结构：

  • 它是指计算机系统的外部结构，包括硬件和软件的组织和交互方式。
  • 计算机体系结构包括指令集、数据表示、存储系统、输入输出系统等。
  • 不同的计算机体系结构有不同的性能和功能。

计算机组成：

  • 是指如何实现计算机体系结构所体现的属性。
  • 它是指计算机系统的内部结构，包括硬件和软件的组织和交互方式。
  • 计算机组成包括CPU、内存、存储系统、输入输出系统等。
  • 不同的计算机组成有不同的性能和功能。

例子：指令系统体现了机器的属性，这是属于计算机结构的问题。但指令的实现如加法指令、减法指令等，是属于计算机组成的问题。

1.4 计算机的基本组成

1.4.1 冯诺依曼计算机的特点

  • 计算机由五大部件组成：运算器、控制器、存储器、输入设备、输出设备。
  • 指令和数据以同等地位存放在存储器中，可按地址访问。
  • 指令和数据以二进制形式表示。
  • 指令由操作码和地址码组成。
  • 指令在存储器中顺序执行，通常指令是顺序执行的。
  • 机器以运算器为中心，输入输出设备与存储器的传输通过运算器完成。
  • 计算机的性能主要由运算器的速度和控制器的速度决定。
