2. Ch1 Python语言入门

2.1 Python 的面世与版本

Python 由 Guido van Rossum 设计，最初是为了创造一种简洁、易读、适合快速开发的脚本语言。

重要版本节点：

  • Python 2.0：2000 年发布，支持完整垃圾回收与 Unicode；
  • Python 3.0：2008 年发布，与 Python 2 不完全兼容；
  • 本课程只按 Python 3 语法复习。

Python 2 与 Python 3
Python 2 中部分语法与 Python 3 不同，例如除法、print 写法、range 行为等。考试以 Python 3 为准，不要混用旧语法。

2.2 Python 解释器

Python 是一门解释型语言。Python 语言规定语法规则，而解释器负责执行符合这些规则的程序。

常见解释器：

  • CPython：最常见的 Python 实现，用 C 语言编写；
  • Jython：运行在 Java 平台上的 Python 实现；
  • IronPython：面向 .NET 与 ECMA CLI；
  • PyPy：重视运行效率的 Python 实现；
  • Zhpy：支持中文语句风格的 Python 实现。

2.3 Python 的特色

Python 的主要特点：

  • 语法简洁，可读性强；
  • 提供交互式运行环境；
  • 变量无需提前声明类型；
  • 通过缩进组织代码块；
  • 内置高级数据结构，如列表、字典、集合；
  • 标准库和第三方库丰富；
  • 跨平台，可用于 Windows、Unix、macOS；
  • 支持面向对象、模块和包；
  • 可扩展、可嵌入，可与 C/C++ 等语言配合。

直观理解
Python 的优势不只是“语法短”，更重要的是它把常用数据结构和库生态做得很顺手，适合快速实验、数据处理、脚本自动化和科学计算。

10. 易错点与复习清单

10.1 易错点

  1. 模块通常就是 .py 文件，包通常是包含 __init__.py 的目录。
  2. import module 后调用函数要写 module.func()。
  3. from module import func 后可以直接写 func()。
  4. import * 容易造成命名冲突，不推荐滥用。
  5. __name__ == "__main__" 用于区分直接运行和被导入。
  6. 类定义用 class，函数/方法定义用 def。
  7. 实例方法第一个参数通常是 self，调用时不需要手动传。
  8. __init__() 会在实例化对象时自动调用。
  9. 私有属性常写成 self.__x，类外不应直接访问。
  10. 子类重写父类方法后，会优先调用子类版本。
  11. super() 用于调用父类方法，常见于子类构造函数中。
  12. *args 收集为元组，**kwargs 收集为字典。
  13. 类对象排序时，key 通常写成 lambda x: x.get_xxx()。
  14. 使用 hashlib.md5() 时输入应为 bytes。
  15. 文件和目录操作前要确认路径，避免覆盖或删除错误文件。

11. 易错点与复习清单

11.1 易错点

  1. Python 列表乘法是重复，NumPy 数组乘法是逐元素乘法。
  2. NumPy 切片常返回视图，修改切片可能影响原数组。
  3. axis=0 通常表示按列统计，axis=1 通常表示按行统计。
  4. 布尔索引用的是布尔数组，不是普通条件文字。
  5. np.dot() 对一维数组是内积，对二维数组是矩阵乘法含义。
  6. 矩阵逐元素乘法用 *，矩阵乘积用 @ 或 np.dot()。
  7. np.savetxt() 保存文本时要注意 fmt 和 delimiter。
  8. np.loadtxt() 读取分隔文本时，分隔符要与文件一致。
  9. pandas 推荐用 loc 和 iloc，避免索引歧义。
  10. df.values 是属性，不是 df.values()。
  11. pd.to_datetime() 解析失败时可用 errors="coerce"。
  12. groupby() 后要明确对哪一列做什么聚合。
  13. plt.show() 通常放在绘图语句之后显示图像。
  14. 画 log(x) 时，x 必须大于 0。
  15. 随机实验若需要复现，要设置 np.random.seed()。
